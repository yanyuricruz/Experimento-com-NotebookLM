---

## Objetivo
Este projeto tem como objetivo experimentar o NotebookLM como uma ferramenta de apoio aos estudos, ajudando a organizar conteúdos e facilitar o aprendizado.

O NotebookLM é um software de anotações com inteligência artificial que analisa fontes fornecidas pelo usuário e gera conteúdos baseados nessas fontes. Ele funciona como um assistente que cria resumos, slides, flashcards, tabelas, guias de aprendizado e explicações.

Como as respostas são fundamentadas nas fontes, a probabilidade de gerar alucinação tende a ser menor em comparação com modelos que respondem baseado nos próprios dados de treinamento.

---

## Conteúdos deste NotebookLM
Este notebook é treinado por fontes sobre o que é IA, como funcionam os modelos de linguagem, um resumo da história da IA e as técnicas de engenharia de prompt. O material é voltado para iniciantes que desejam aprender os fundamentos de IA.
